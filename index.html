My Projects

Saarthi: Driver Alertness Monitoring Device

An IoT-enabled device designed to prevent drivers from falling asleep, ensuring safety on the road.

→

Saarthi is an IoT-enabled device that monitors driver alertness to prevent accidents caused by drowsiness. Using a camera to detect eye movement, the system triggers an alarm if the driver's eyes are closed for more than 3 seconds. Built with a Raspberry Pi and custom coding, Saarthi aims to reduce road accidents and save lives by ensuring drivers stay alert.

Nagrik Aur Samvidhan: Interactive Constitution Learning Game

An interactive, choice-based game that helps players learn about the Indian Constitution in a fun and engaging way.

→

Nagrik Aur Samvidhan is an interactive game that educates players about the Indian Constitution through choice-based scenarios. Created for the Smart India Hackathon, the game presents players with decisions based on the Constitution, making learning fun and immersive. Its success lies in its engaging plot and ability to teach constitutional principles in an accessible way.

ProjeXStock: AI-Based Stock Price Prediction

A web application that predicts stock and share prices using AI APIs to help users avoid losses in the stock market.

→

ProjeXStock is a web application that predicts stock prices using AI APIs like Alpha Vantage. The platform analyzes market trends to provide users with insights, helping them make informed investment decisions and reduce risks. Though still a work in progress, ProjeXStock aims to offer a valuable tool for navigating the stock market.

Anti Bunk Attendance System: Future Project

A future system that uses IP addresses to verify student attendance, preventing bunking in college.

→

The Anti Bunk Attendance System uses IP addresses to verify student attendance and prevent bunking. Students are granted attendance when they visit a link while connected to the college Wi-Fi. If accessed from another network, the system detects the bunk and prevents attendance. This project aims to improve accountability in college attendance systems.
